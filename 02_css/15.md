<h1>Sombras e Filtros</h1>

<br />

<h2>1. Sombras</h2>  



As sombras em CSS são utilizadas para adicionar profundidade e realismo aos elementos de uma página. Elas ajudam a criar a sensação de que um elemento está elevado em relação ao plano de fundo, tornando o layout mais dinâmico e agradável visualmente.

Existem dois principais tipos de sombras em CSS:

* **Sombras de caixa** (`box-shadow`)
* **Sombras de texto** (`text-shadow`)

Ambas seguem uma lógica semelhante de definição de deslocamento, desfoque e cor, mas se aplicam a contextos diferentes.

<br />

<h3>1.1. Box Shadow</h3>  



A propriedade `box-shadow` é usada para adicionar uma ou mais sombras em torno de um elemento de caixa, como uma `<div>`, um botão ou uma imagem.

A sintaxe básica da propriedade é a seguinte:

```css
box-shadow: deslocamentoX deslocamentoY desfoque cor;
```

Vamos entender cada parte:

* **deslocamentoX (offset-x):** define a posição horizontal da sombra. Valores positivos movem a sombra para a direita e negativos para a esquerda.
* **deslocamentoY (offset-y):** define a posição vertical da sombra. Valores positivos movem a sombra para baixo e negativos para cima.
* **desfoque (blur-radius):** controla o nível de desfoque da sombra. Quanto maior o valor, mais suave e difusa será a sombra.
* **cor (color):** define a cor da sombra.

Na imagem abaixo, vemos alguns exemplos de uso da propriedade **`box-shadow`**:

<div align="center"><img src="https://imgur.com/8ITeSZX.png" title="source: imgur.com" /></div>  

- **`box-shadow: 5px 5px 10px gray;`** → Cria uma sombra cinza deslocada **para a direita (5px)** e **para baixo (5px)**, com **desfoque de 10px**.
- **`box-shadow: -5px -5px 5px gray;`** → Cria uma sombra cinza deslocada **para a esquerda (-5px)** e **para cima (-5px)**, com **desfoque de 5px**.
- **`box-shadow: 0px 0px 5px gray;`** → Cria uma sombra cinza **sem deslocamento**, apenas **um brilho suave ao redor** do elemento.
- **`box-shadow: -5px 5px 10px gray;`** → Cria uma sombra cinza deslocada **para a esquerda (-5px)** e **para baixo (5px)**, com **desfoque de 10px**.

<br />   

<h4>1.1.2. Sombras internas</h4>  



Por padrão, as sombras são aplicadas externamente ao elemento. No entanto, é possível criar sombras internas utilizando o valor `inset`.

O valor `inset` inverte a direção da sombra, fazendo com que ela seja aplicada para dentro da caixa, criando o efeito de relevo invertido.

Na imagem abaixo, vemos exemplos que ilustram a diferença entre o uso da propriedade **`box-shadow`** em seu modo padrão e com o valor `inset`, que cria o efeito de sombra interna.

<div align="center"><img src="https://imgur.com/6tXxvbD.png" title="source: imgur.com" /></div>  

- **`box-shadow: 10px 10px 5px gray;`** → Sombra **externa**, deslocada **para a direita (10px)** e **para baixo (10px)**, com **desfoque de 5px**.
- **`box-shadow: -10px -10px 5px gray;`** → Sombra **externa**, deslocada **para a esquerda (-10px)** e **para cima (-10px)**, com **desfoque de 5px**.
- **`box-shadow: inset 10px 10px 5px gray;`** → Sombra **interna**, projetada **para dentro**, deslocada **para a direita (10px)** e **para baixo (10px)**, com **desfoque de 5px**.
- **`box-shadow: inset -10px -10px 5px gray;`** → Sombra **interna**, projetada **para dentro**, deslocada **para a esquerda (-10px)** e **para cima (-10px)**, com **desfoque de 5px**.

<h3>1.2. Text Shadow</h3>  



A propriedade `text-shadow` adiciona sombra ao texto, criando efeitos de destaque, profundidade ou iluminação.

A sintaxe é semelhante à da propriedade `box-shadow`:

```css
text-shadow: deslocamentoX deslocamentoY desfoque cor;
```

Na imagem abaixo, vemos alguns exemplos de uso da propriedade **`text-shadow`**:

<div align="center"><img src="https://imgur.com/PX6GV9E.png" title="source: imgur.com" /></div>  

- **`text-shadow: 1px 1px 2px black;`** → Cria uma sombra **preta** no texto, deslocada **1px para a direita** e **1px para baixo**, com **desfoque de 2px**.
- **`text-shadow: -1px -1px 2px black;`** → Cria uma sombra **preta** no texto, deslocada **1px para a esquerda** e **1px para cima**, com **desfoque de 2px**.
- **`text-shadow: 0px 0px 2px black;`** → Cria uma sombra **preta no texto sem deslocamento**, formando **um brilho suave** ao redor do texto.
- **`text-shadow: -1px 1px 2px black;`** → Cria uma sombra **preta** no texto, deslocada **1px para a esquerda** e **1px para baixo**, com **desfoque de 2px**.

<br />

> [!IMPORTANT]
>
> Sombras de texto não possuem o modificador `inset` e sempre seguem o formato padrão de sombra externa.

<h2>2. Filtros</h2>



Os **Filtros (CSS Filters)** são recursos que permitem **aplicar efeitos visuais diretamente em elementos HTML**, principalmente imagens, vídeos e backgrounds, sem a necessidade de editar arquivos em ferramentas externas.

Com filtros, é possível ajustar brilho, contraste, saturação, desfoque e outros efeitos visuais **apenas com CSS**, de forma dinâmica e reversível. Além disso, em imagens SVG é possível alterar a cor dinamicamente.

> Pense nos filtros de aplicativos de fotos. O CSS Filters funciona de forma parecida, mas aplicado diretamente no navegador.

Os filtros CSS possuem algumas características importantes:

- São aplicados **via código**, sem alterar o arquivo original
- Funcionam em **imagens, vídeos e qualquer elemento renderizado**
- Podem ser **combinados** em um único efeito
- São **dinâmicos**, podendo mudar com interações ou estados
- Facilitam ajustes visuais rápidos no layout

Eles são muito usados para efeitos de hover, estados desativados, destaques visuais e ajustes estéticos simples.

Para trabalhar com filtros em CSS, utilizamos a propriedade `filter` em conjunto com as funções de filtro.

<img src="https://i.imgur.com/DSBasbH.png" title="source: imgur.com" width="4%"/>**CSS - Exemplo:**

```css
img {
  filter: grayscale(100%);
}
```

Nesse caso, a imagem é exibida em tons de cinza, sem que o arquivo original seja modificado.

<br />

> [!WARNING]
>
> A propriedade `filter` também pode impactar performance em dispositivos mais antigos, especialmente quando aplicada em animações ou grandes imagens.

<br />

A propriedade `filter` aceita várias **funções**, cada uma responsável por um tipo de efeito visual, como veremos a seguir:

<br />

### ✅ `blur()`

Aplica um **desfoque** ao elemento.

```css
img {
  filter: blur(4px);
}
```

**Uso comum:**

- Imagens de fundo
- Conteúdo secundário
- Estados de carregamento

<br />

### ✅ `brightness()`

Controla o **brilho** do elemento.

```css
img {
  filter: brightness(120%);
}
```

**Valores:**

- Menor que `100%` → escurece
- Maior que `100%` → clareia

<br />

### ✅ `contrast()`

Ajusta o **contraste** entre claro e escuro.

```css
img {
  filter: contrast(150%);
}
```

É útil para dar mais destaque visual a imagens ou fundos.

<br />

### ✅ `grayscale()`

Remove a cor, deixando o elemento em **tons de cinza**.

```css
img {
  filter: grayscale(100%);
}
```

**Uso comum:**

- Efeito visual em galerias
- Destaque apenas no hover
- Conteúdos desativados

<br />

### ✅ `sepia()`

Aplica um efeito de **tom envelhecido**, semelhante a fotos antigas.

```css
img {
  filter: sepia(80%);
}
```

Muito usado para efeitos decorativos e estéticos.

<br />

### ✅ `saturate()`

Controla a **intensidade das cores**.

```css
img {
  filter: saturate(150%);
}
```

Valores baixos deixam a imagem mais “lavada”; valores altos intensificam as cores.

<br />

### ✅ `invert()`

Inverte as cores do elemento.

```css
img {
  filter: invert(100%);
}
```

**Uso comum:**

- Efeitos específicos
- Ajustes rápidos em ícones
- Experimentos visuais

<br />

### ✅ `opacity()`

Controla a **transparência**, semelhante à propriedade `opacity`, mas aplicada como filtro.

```css
img {
  filter: opacity(70%);
}
```

Pode ser útil quando combinado com outros filtros.

<br />

### ✅ `hue-rotate()`

Gira as cores no espectro de matiz.

```css
img {
  filter: hue-rotate(90deg);
}
```

É um filtro mais avançado, usado para efeitos criativos ou ajustes rápidos de cor.

<br />

<h3>2.1. Combinando múltiplos filtros</h3>




Uma das maiores vantagens do `filter` é poder **combinar vários efeitos** em uma única linha.

```css
img {
  filter: grayscale(100%) brightness(120%) contrast(110%);
}
```

Os filtros são aplicados **da esquerda para a direita**, e pequenas mudanças já alteram bastante o resultado final.

<br />

> Não se preocupe em decorar as funções da propriedade filter.
>
> Existem diversas **ferramentas on-line** que permitem criar filtros de forma visual e interativa, facilitando a geração de código CSS capazes de reproduzir efeitos dignos do Photoshop em imagens e até gerar uma cor específica por meio de **CSS Filters**.
>
> Essas ferramentas são especialmente úteis, pois possibilitam aplicar efeitos e cores dinamicamente sem a necessidade de editar o arquivo original, mantendo flexibilidade e reaproveitamento das imagens.
>
> Ao final deste material, estarão disponíveis **nos anexos I e II, links** para essas ferramentas, permitindo que você explore, teste combinações de filtros e gere o código CSS de forma prática.

<br />

A seguir, veremos um exemplo prático de **filtros CSS aplicados a uma imagem**, demonstrando como a combinação de filtros altera visualmente a imagem sem modificar o arquivo original.

<img src="https://i.imgur.com/O5NjoiA.png" title="source: imgur.com" width="5%"/>**Exemplo:**

```html
<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Página Exemplo</title>
  </head>
  <body>
      <img src="https://i.imgur.com/Fq6kGS8.jpg" alt="Paisagem">
      <img class="filtro_01" src="https://i.imgur.com/Fq6kGS8.jpg" alt="Paisagem">
      <img class="filtro_02" src="https://i.imgur.com/Fq6kGS8.jpg" alt="Paisagem">
  </body>
</html>

```

<img src="https://i.imgur.com/DSBasbH.png" title="source: imgur.com" width="4%"/>**CSS - Exemplo:**

```css
img{
  width: 30%;
}

.filtro_01{
   filter: brightness(88%) grayscale(69%) sepia(70%);
}

.filtro_02{
  filter: hue-rotate(30deg) invert(25%) saturate(81%);
}
```

<img src="https://i.imgur.com/3HPZ0By.png" title="source: imgur.com" width="3%"/>**Resultado:**

<div align="center"><img src="https://i.imgur.com/SajjIGB.png" title="source: imgur.com"/></div>

No exemplo acima, foram aplicados os seguintes filtros:

- A primeira imagem é a imagem original, sem filtros
- Na segunda imagem, foi aplicado o **`.filtro_01`**, que reduz levemente o brilho da imagem e remove parte das cores originais, aplicando um tom amarelado típico do efeito sépia. O resultado é uma aparência mais opaca e envelhecida.
- Na terceira imagem, foi aplicado o **`.filtro_02`**, que desloca as cores no círculo cromático (mudando a cor base da imagem), inverte parcialmente os tons e diminui a intensidade das cores. O efeito final altera a paleta da imagem de forma sutil e estilizada.

<br />

## Boas práticas



- Use filtros com **moderação**
- Evite exageros que prejudiquem a legibilidade
- Teste em diferentes tamanhos de tela
- Combine filtros para efeitos sutis
- Lembre-se de que filtros são recursos visuais, não funcionais

O objetivo do filtro é complementar o design, não chamar mais atenção que o conteúdo.
